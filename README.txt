Call to action bar on mobile 

== Description ==
Show action bar. Easy for call to action on mobile site.


== Installation ==


1. Upload `mobi-actionbar.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to Setting -> Mobile Action Bar
